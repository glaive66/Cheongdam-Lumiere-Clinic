2. [청담 루미에르 의원] UI 디자인 컨셉 및 스타일 가이드
1. 컬러 팔레트 (Color Palette) : 호텔 스파의 온기와 고급스러움

Background White (#FCFCFC): 순백의 화이트가 아닌 아주 미세하게 따뜻한 톤이 감도는 화이트를 전체 배경으로 사용하여, 눈의 피로를 덜고 시원한 여백의 미를 돋보이게 합니다.

Sub Beige (#E6DCD3): 메뉴바, 섹션 구분 배경, 이미지 테두리 등에 서브 베이지를 활용하여 피부과의 차분하고 안정적인 분위기를 연출합니다.

Main Gold (#B89B72): 카카오톡 상담, 네이버 예약 플로팅 버튼, 주요 강조 카피(티타늄 리프팅 장점 등), 버튼 클릭 시의 호버(Hover) 효과 등에 메인 골드를 포인트 컬러로 사용하여 럭셔리한 정체성을 부여합니다.

2. 타이포그래피 (Typography) : 우아함과 전문성의 조화

메인 폰트: 얇고 세련된 명조체(Serif) 계열을 웹 폰트로 적용합니다. 이는 빛을 형상화한 루미에르의 얇은 다이아몬드 라인 로고와 완벽한 조화를 이루며, 하이엔드 뷰티 브랜드 특유의 우아하고 신뢰감 있는 이미지를 전달합니다.

가독성 확보: 상세 설명이나 안내 문구에는 굵기 조절이 용이한 산세리프(고딕) 계열을 서브로 매칭하여, 모바일 환경에서도 연령대와 무관하게 뛰어난 가독성을 제공합니다.

3. 비주얼 및 인터랙션 (Visual & Interaction)

여백의 미 (White Space): 각 섹션(히어로 배너, 의료진 소개, 전후 사진 등) 사이의 간격을 넓고 시원하게 배치하여, 숨 막히는 정보 전달이 아닌 갤러리 작품을 감상하는 듯한 여유로움을 선사합니다.

트랜지션 효과 (Soft Transition): 페이지를 스크롤할 때 텍스트와 이미지가 아래에서 위로 부드럽게 나타나는 '페이드 인(Fade-in)' 효과를 적용합니다. 시각적 피로도를 낮추는 동시에 프리미엄 피부과만의 정제되고 세련된 느낌을 강조합니다.
